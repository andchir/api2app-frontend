# Создание мини-приложений ВКонтакте

VK Mini Apps — это открытая платформа встраиваемых кросс-платформенных приложений, которые расширяют возможности сайта и приложения ВКонтакте. Для разработки используются стандартные веб-технологии: HTML, JavaScript, CSS.

https://dev.vk.com/ru/mini-apps/overview

1. В соцсети ВКонтекте откройте раздел "Разработчикам" (ссылка слева внизу) или перейдите по ссылке: https://dev.vk.com/ru.

2. Нажмите кнопку "Создать приложение". Введите название приложения, выберите категорию и нажмите кнопку "Создать".

3. Для настройки следуйте документации https://dev.vk.com/ru/mini-apps/settings/general/information

4. В разделе "Размещение" Вы можете использовать URL встраивания Вашего приложения (раздел "Приложения" -> "Мои приложения" -> "Поделиться" на сайте api2app).

![ВК Скриншот 1](https://api2app.org/media/uploads/1730054491_ksnip_20241027-213539.png)

![ВК Скриншот 2](https://api2app.org/media/uploads/1730056257_ksnip_20241027-214605.png)

5. Также Вы можете скачать приложение и использовать [хостинг ВК](/ru/docs#using-vk-hosting) для публикации.

# Создание платных подписок

Для монетизации приложения в ВК можно продавать виртуальные товары и услуги за голоса (валюта ВКонтакте).

Подробнее:  
https://dev.vk.com/ru/mini-apps/monetization/payments

1. Добавить элемент "Подписка пользователя" в приложение.

![Скриншот 1](https://api2app.org/media/uploads/1729675311_screenshot_20241023-120233.png)

2. В свойствах можно изменить текст кнопки, иконку и идентификатор подписки.

3. Идентификатор подписки - это идентификатор подписки, которая создана в системе api2app. На данный момент нет возможности создавать свои собственные подписки, а только использовать уже созданные.

![Скриншот 2](https://api2app.org/media/uploads/1729675245_vk-subs.png)

4. Введите идентификатор, например, "remove_ad" ([полный список](/ru/docs#subscriptions-vk-list)). Эта подписка даёт пользователю приложения ВК отключить рекламу на месяц (стоимость: 10 голосов). По умолчанию во всех приложениях включены два рекламных баннера: маленький вверху страницы приложения и большой, который появляется на время по середине экрана после нажатия на кнопку в приложении.

Если Вы хотите сделать полностью платным своё приложение, то в настройках API нужно включить параметр **"Только для платных подписчиков"** и использовать один из идентификаторов с префиксом **"main_"**.

5. Нажмите кнопку "Настройки" в режиме редактирования приложения. Заполните поля "ID приложения ВК" и "Защищённый ключ приложения ВК". Это необходимо для проверки подписи запросов ВКонтакте при проведении платежей.

![Скриншот 3](https://api2app.org/media/uploads/1729675270_screenshot_20241023-121146.png)

# Список доступных подписок для ВКонтакте

Период действия подписки: 30 дней, пробный период: 3 дня.

**remove_ad** - Отключить рекламу. Стоимость: 10 голосов.

**main_10** - Основной функционал. Стоимость: 10 голосов.

**main_20** - Основной функционал. Стоимость: 20 голосов.

**main_30** - Основной функционал. Стоимость: 30 голосов.

**premium_20** - Отключение рекламы и улучшение функций. Стоимость: 20 голосов.

**premium_30** - Отключение рекламы и улучшение функций. Стоимость: 30 голосов.

# Использование хостинга статики ВК

Для хранения и публикации файлов приложения можно использовать бесплатный хостинг файлов ВКонтакте.

Документация:  
https://dev.vk.com/ru/mini-apps/development/hosting/overview

1. Скачать приложение в разделе "Приложения" -> "Мои приложения".

2. Создать папку на жестком диске своего компьютера, создать в ней ещё одну папку с названием "build" и распаковать в ней архив приложения.

3. Запустить терминал (Windows PowerShell, или Командyную строку - cmd). Перейти в папку, в которой находится папка "build" с файлами.
```
cd <путь-к-папке-приложения>
```  

4. Если в этой папке нет файла **package.json**, инициализировать пакет npm (можно принять значения всех параметров по умолчанию).
```
npm init
```  

5. Установите библиотеку **vk-miniapps-deploy**:
```
npm install @vkontakte/vk-miniapps-deploy --include=dev
```  

6. В текущей папке создать файл **vk-hosting-config.json** с таким содержимым:
```json
{
  "static_path": "build",
  "app_id": 123456,
  "endpoints": {
    "mobile": "index.html",
    "web": "index.html",
    "mvk": "index.html"
  }
}
```  

**static_path** (build) - папка, в которой находятся файлы приложения.  
**app_id** - Идентификатор приложения. Расположен в настройках мини-приложения (в ВКонтакте), на вкладке **Настройка** — значение **ID приложения**.

7. Добавить в файл **package.json** скрипты **deploy** и **predeploy**:
```json
"scripts": {
    ...
    "build": "...",
    "predeploy": "npm run build",
    "deploy": "vk-miniapps-deploy"
}
```
Код из примера выше предполагает, что в вашем проекте определён скрипт **build**, который будет вызван скриптом **predeploy**. Иначе в **build** можно ввести любую команду, например, **pwd** (вывести путь текущей папки).

8. Добавить в файл **package.json** поле "homepage": ".".
```json
"scripts": {
    ...
},
"homepage": "."
```

9. Выполнить следующую команду. Она запустит процедуру размещения файлов на хостинге:
```
npm run deploy
```

![vk-deploy-1](https://api2app.org/media/uploads/1730662272_vk-miniapps-deploy-1.png)

Если Вы выкладываете файлы на хостинг впервые, библиотеке необходимо получить специальный ключ доступа для выполнения этой задачи.

Скопируйте ссылку, откройте её в браузере и воспользуйтесь инструкциями на экране.

![vk-deploy-2](https://api2app.org/media/uploads/1730662286_vk-miniapps-deploy-2.png)

Вернитесь в окно командной строки и нажмите y или Enter, чтобы подтвердить, что вы сформировали ключ.

Сформированный ключ появится в окне командной строки после завершения процедуры.

![vk-deploy-3](https://api2app.org/media/uploads/1730662304_vk-miniapps-deploy-3.png)

После окончания загрузки в окне командной строки появятся URL следующего вида:

![vk-deploy-4](https://api2app.org/media/uploads/1730662317_vk-miniapps-deploy-4.png)

**Ограничения**:
- Выкладывать файлы на хостинг ВКонтакте можно не более 24 раз в сутки.
- При повторном размещении файлов предыдущая версия не удаляется.
- Для каждого приложения хостинг хранит до 10 production-версий и до 100 staging-версий.
- Максимальный размер промежуточного ZIP-архива — 300 Мбайт.
- Размещение аудио- и видеофайлов не допускается.

# Создание чат-бота для сообщества ВКонтакте

1. Перейти на страницу своего сообщества ВК, открыть раздел "Управление" -> "Дополнительно" -> "Работа с API". нажать кнопку **"Создать ключ"**, отметить минимум два верхних пункта. Закончить создание ключа.  
   ![](https://api2app.org/media/uploads/1757774694_vk_chatbot_002.png)  
   ![](https://api2app.org/media/uploads/1757774683_vk_chatbot_001.png)
2. Скопировать ключ в буфер обмена. Открыть приложение на сайте **api2app**, нажать кнопку "Настройки", в появившемся окне вставить ключ в поле **"Ключ доступа сообщества ВК"**.
3. Нажать кнопку "Сохранить". При сохранении приложения будет создан сервер для **Callback URL** для сообщества ВК, далее нужно подтвердить это действие. В настройках сообщества перейти в раздел "Callback URL", скопировать строку, которую должен вернуть сервер.  
   ![](https://api2app.org/media/uploads/1757774704_vk_chatbot_003.png)
4. В приложении на сайте **api2app** добавить элемент **"Текстовое поле"**. В параметрах этого поля изменить имя на **"confirmation"**. В значение вставить скопированную ранее строку. Сохранить изменения приложения. Убедитесь, что приложение опубликовано (может быть скрытым).
5. На сайте ВК в разделе "Callback URL" нажать кнопку **"Подтвердить"**. Должно появиться сообщение "Адрес сервера успешно сохранён".
6. На сайте ВК перейти в раздел **"Сообщения"** -> **"Настройки для бота"**. Включить параметр **"Добавить кнопку «Начать»"**. Сохранить.
7. Для создания начального меню нужно использовать элемент **"Выбор (select)"** с именем **"start"**. Создайте значения, эти значения будут использованы для создания кнопок меню чат-бота. можно использовать эмодзи (смайлики и иконки).  
   ![](https://api2app.org/media/uploads/1757774716_vk_chatbot_004.png)
8. Можно протестировать меню, нажав кнопку **"Начать"** в чате сообщества.  
   ![](https://api2app.org/media/uploads/1757774965_vk_chatbot_005.jpg)
9. Для того, чтобы при нажатии на кнопки меню появлялась текстовая информация, нужно в приложении добавить текстовые элементы с соответствующими названиями. Например, если в начальном меню есть пункт **"Услуги"**, необходимо добавить элемент с типом **"Текст"** и названием поля "Услуги".
10. Можно также использовать элемент "Выбор (select)", в этом случае чат-бот выведет **инлайн кнопки**. Такие кнопки могут быть использованы, например, для выбора даты или времени при заказе услуги.  
    ![](https://api2app.org/media/uploads/1757775739_vk_chatbot_007.png)
11. Если необходимо, чтобы чат-бот выводил изображение (или видео), нужно это изображение сначала загрузить в сообщество. Открыть это изображение и скопировать в адресной строке ключ медиавложения в формате "{type}{owner_id}_{media_id}". Пример: **photo-232611602_457239017**.  
    ![](https://api2app.org/media/uploads/1757775494_vk_chatbot_006.png)  
    Затем использовать эту строку как значение текстового элемента, который будет использоваться как ответ чат-бота.  
    Смотрите видео-урок [Создание чат-бота для сообщества ВК](/ru/docs?tab=3).
